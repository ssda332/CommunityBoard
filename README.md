# 간단한 게시판 프로젝트

## 개발 목표

* 게시판 기능(게시판 CRUD, 댓글 CRUD, 게시글 검색, 대댓글 등등) 구현
* 회원 기능(로그인, 회원가입, 마이페이지) 구현
* 관리자 기능(회원, 게시글 관리) 구현


## 개발 환경

### Language
* Spring-Framework
* Spring Security

### Template Engine
* Thymeleaf

### View
* HTML5, CSS3, Javascript

### Web server
* Apache Tomcat 8.5

### DBMS
* Oracle, AWS RDS Oracle

### Library
* Jquery
* Mybatis

### Server Hosting
* AWS EC2


## 구현 기능
* 게시글 CRUD, 댓글 CRUD, 게시글 검색
* 스프링 시큐리티를 이용한 로그인, 회원가입, 로그인 검증
* 클라우드 서버를 통한 웹 서비스 배포
